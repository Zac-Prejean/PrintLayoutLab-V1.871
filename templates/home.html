Welcome to the Home Page

The PreCheck page handles CSV files containing product information and personalization data. It allows users to upload a CSV file, preview and edit the personalization fields, validate SKUs, and download an updated CSV file with the changes. Go to PreCheck Page

The Designer page processes a CSV file containing product information and generates personalized images for each product based on the user's input. The images are saved to the user's 'Downloads' folder. Go to Designer Page

The Template Merger Page is for processing and combining multiple images loaded by the user. It also provides a preview for the final combined image and allows the user to download it. Go to Template Merger Page
